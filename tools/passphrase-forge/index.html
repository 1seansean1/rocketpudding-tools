Everything runs client-side in your browser. Secrets are drawn with crypto.getRandomValues (a cryptographically secure RNG — never Math.random) and selected by rejection sampling, so every word and character is uniform with no modulo bias. Nothing is generated on a server, stored, logged, or transmitted. The only external request this page makes is for the IBM Plex fonts.
Passphrases use the full EFF large wordlist (7,776 words, one per five-dice roll) published by the Electronic Frontier Foundation under CC-BY. Each word contributes log2(7776) ≈ 12.925 bits. Crack time assumes an offline attacker at 1012 guesses per second against the average (half) of the guess space; it credits only randomness, not any secret transform. Learn more at eff.org/dice.
