method & honest approximations

Source is parsed with acorn. Statements are grouped into basic blocks; the blocks are one node set carrying three edge layers: CFG (solid, T/F branch labels, loop back-edges), DFG (dashed, per-variable reaching-definition def→use edges) and AST containment (dotted, block → nearest enclosing construct). This is a deliberately small approximation of a Joern-style CPG:
