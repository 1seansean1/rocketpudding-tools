Regex cheat sheet

Character classes

.any char except newline (with s flag: any char)
\ddigit [0-9]
\Dnon-digit
\wword char [A-Za-z0-9_]
\Wnon-word char
\swhitespace (space, tab, newline…)
\Snon-whitespace
[abc]any one of a, b, c
[^abc]any char except a, b, c
[a-z0-9]character ranges (combinable)
\p{L}Unicode property, e.g. any letter (needs u flag); \P{…} negates

Anchors & boundaries

^start of string (with m: start of each line)
$end of string (with m: end of each line)
\bword boundary (between \w and \W or string edge)
\Bnot a word boundary

Quantifiers

*0 or more (greedy)
+1 or more (greedy)
?0 or 1 (greedy)
{n}exactly n
{n,}n or more
{n,m}between n and m
*? +? ??lazy — append ? to any quantifier to match as little as possible
{n,m}?lazy bounded repeat

Groups & backreferences

(x)capturing group, numbered 1…n by order of opening paren
(?<name>x)named capturing group (also gets a number)
(?:x)non-capturing group
a|balternation — a or b (lowest precedence; group to scope)
\1backreference — match group 1’s text again
\k<name>backreference to a named group
$& $1 $<name>in replacement strings: whole match / group n / named group; also $` (before), $' (after), $$ (literal $)

Lookaround (zero-width)

(?=x)lookahead — followed by x
(?!x)negative lookahead — not followed by x
(?<=x)lookbehind — preceded by x
(?<!x)negative lookbehind — not preceded by x

Flags

gglobal — find all matches, not just the first
iignoreCase — case-insensitive
mmultiline — ^ $ match at every line break
sdotAll — . also matches newlines
uunicode — code-point semantics, enables \p{…} and \u{…}
ysticky — match must start exactly at lastIndex

Escapes

\. \* \+ \?literal metacharacters — also \( \) \[ \] \{ \} \| \^ \$ \\ \/
\n \r \tnewline, carriage return, tab (also \0 \f \v)
\xhhchar by 2-digit hex code, e.g. \x41 = A
\uhhhhchar by 4-digit hex code point, e.g. é = é
\u{hhhhh}any code point (needs u flag), e.g. \u{1F600}
zero dependencies
