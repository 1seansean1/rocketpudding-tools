Known limitations

No remote / file $ref — only local JSON-pointer refs (#/definitions/x) resolved within the same document.
No draft 2019-09 / 2020-12 dynamic refs — no $dynamicRef, $recursiveRef, $anchor, or unevaluatedProperties/Items.
format is assertive here — treated as a validation error, unlike the spec default where it is an annotation. Best-effort regexes, not full RFC parsers.
$ref siblings ignored — per draft-07, when $ref is present other keywords in the same object are skipped.
No contentMediaType/Encoding, no $id base-URI resolution, no definitions validation of unreferenced entries.
